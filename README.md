# English Learn Hub

## Proje Tanımı
Kullanıcı dostu İngilizce öğrenme platformu. Seviye tespiti, modüller (kelime kartları, çevirme, günlük cümleler, hikayeler, oyunlar, AI chat) ve ilerleme takibi içerir. Cross-platform (web + Android hazır).

## Modüller
- **Vocabulary Flashcards**: Görsel kelime kartları + telaffuz.
- **Flip Cards**: Çevirme kartları (Türkçe → İngilizce).
- **Daily Sentences**: Günlük cümleler + çeviri + sesli okuma.
- **Stories**: Seviyeye uygun hikayeler + anlama soruları.
- **Games**: Word Snake, Grammar Race, Listening Gap (yeni).
- **AI Chat**: Hata düzelten sohbet botu.
- **Quiz & Sentence Builder**: Mevcut entegrasyon.

## Teknoloji
- Frontend: HTML/CSS/JS (Tailwind opsiyonel).
- Veri: JSON.
- Telaffuz: Web Speech API.
- İlerleme: LocalStorage + Firebase (gelecek).
- Deploy: Vercel.

## Kurulum
1. `git clone https://github.com/ABDULLAH-GOKCEK/englishlearnhub.git`
2. `npm install` (eğer package.json varsa).
3. `index.html` açın veya Vercel'e deploy edin.

## İlerleme Sistemi
- Modül tamamlama: 90+ puan.
- Seviye bitirme: Sertifika (PDF).
- Tam mezuniyet: Üstün başarı belgesi.
